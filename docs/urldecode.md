# 简介
`urldecode` 该命令用来对URL编码的数据进行解码。

# 格式
```
qshell urldecode <DataToDecode>
```

# 帮助文档
可以在命令行输入如下命令获取帮助文档：
```
// 简单描述
$ qshell urldecode -h 

// 详细文档（此文档）
$ qshell urldecode --doc
```

# 鉴权
无

# 参数
- DataToDecode：待解码字符串。 【必须】

# 示例
```
$ qshell urldecode %E5%A4%A7%E6%95%B0%E6%8D%AE%E6%97%B6%E4%BB%A3
大数据时代
```
